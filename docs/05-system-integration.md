## 2. 核心集成点

### 2.1 RaptorSystem 编排器

`RaptorSystem` 类作为中央编排器，实例化并协调所有 25+ 个模块。通过 `SystemConfig` 和 `SystemState` 数据类管理全局配置与运行时状态。

```python
# 系统初始化
system = RaptorSystem(SystemConfig(uav_id="raptor-01"))
system.initialize()

# 设置任务
system.set_mission(waypoints)

# 导航控制
nav_result = system.navigate()

# 避障决策
avoid = system.compute_avoidance(detections, altitude)
```

### 2.2 UAVSimulator 模拟器

提供无硬件的运行时环境验证系统行为，支持物理模拟（速度/加速度/姿态耦合）和电池消耗模拟。

```python
sim = UAVSimulator(SimConfig(dt=0.05))
sim.reset()
for _ in range(100):
    sim.step(throttle=0.5, roll=0.0, pitch=0.3)
```

### 2.3 模块间数据流

```
传感器 → VIO → 位置估计 → Path Planner → Waypoint → MPC/PID → Motor Mixer → 执行器
                 ↓
            1. Obstacle Detector → 避障向量 → 路经修正
            2. Edge Detector    → 目标列表 → NMS → 决策
            3. Network Monitor  → Link Manager → 链路选择
                 ↓
            Swarm Coordinator → Consensus → 集群决策
            Intent Predictor  → 协作预判 → 编队调整
            Evolutionary Game → 任务分配 → 角色切换
```
